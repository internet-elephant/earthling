## Apple Frameworks

Earthling uses the following Apple-provided frameworks, which require no additional attribution:

- **MapKit** — location autocomplete, GPS coordinate capture, and map view
- **SwiftUI** — user interface
- **CoreGraphics** — PDF generation
- **Foundation** — data persistence and file management
